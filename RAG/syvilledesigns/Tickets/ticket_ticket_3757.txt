Customer Service answer: Caro <name>,

Obrigado por entrar em contato. Com a atualização para o Chrome 102.0, é possível que algumas extensões precisem ser atualizadas também. Por favor, tente os seguintes passos:

1. Abra o Chrome e vá para 'chrome://extensions/' para verificar se há atualizações disponíveis para suas extensões. Atualize se solicitado.
2. Desative e reative as extensões para atualizar suas configurações.
3. Limpe o cache do seu navegador para evitar dados em cache conflitantes.
4. Reinicie o seu navegador após concluir essas etapas.

Se o problema persistir, considere entrar em contato com o desenvolvedor da extensão para atualizações de compatibilidade. Informe-nos se precisar de mais assistência.
