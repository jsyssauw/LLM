Customer Service answer: Bonjour <name>,

Merci de nous avoir contactés. Veuillez essayer les étapes suivantes pour résoudre le problème de chargement des extensions de Visual Studio Code :

1. Redémarrez Visual Studio Code.
2. Désactivez toutes les extensions et activez-les une par une pour identifier celles qui posent problème.
3. Videz le cache des extensions depuis la palette de commandes en utilisant 'Effacer l'historique de l'éditeur'.
4. Vérifiez les mises à jour supplémentaires ou réinstallez Visual Studio Code si nécessaire.

Faites-nous savoir si vous avez besoin de plus d'assistance.
