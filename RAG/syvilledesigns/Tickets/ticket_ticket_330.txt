Merci d'avoir contacté concernant votre problème avec IntelliJ IDEA. Veuillez essayer les étapes suivantes : 

1. Videz le cache de l'IDE via Fichier > Invalider les caches / Redémarrer.
2. Vérifiez les plugins tiers qui pourraient entrer en conflit et désactivez-les temporairement.
3. Mettez à jour votre JDK Java vers la dernière version supportée par IntelliJ IDEA.
4. Si possible, essayez de revenir temporairement sur les modifications récentes du code pour isoler le problème.

Si le problème persiste après ces étapes, veuillez contacter le support de JetBrains pour une solution plus dédiée ou vérifier si des correctifs sont disponibles sur leur site.
